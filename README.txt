# Execute o aplicativo da Oficina
O aplicativo da oficina foi desenvolvido em Python e pode ser executado em qualquer sistema operacional que possua o interpretador Python instalado.

## Pré-requisitos
Antes de começar, verifique se você tem os seguintes itens:

- Interpretador Python instalado (pode ser baixado em https://www.python.org/)
- Arquivo Main.py baixado e descompactado em uma pasta de sua preferência
- Tem instalado todos os requesitos presentes em 'requirements.txt'
    - para instalar 'pip install -r requirements.txt' 

## Execute o programa
Para executar o programa, siga os seguintes passos:

1. Abra o terminal do seu sistema operacional
2. Navegue até a pasta onde o arquivo 'Main.py' está localizado e use o comando 'cd'
3. Escreva o comando 'python Main.py' e clique Enter

O programa será iniciado e poderá utilizar o aplicativo da oficina.
